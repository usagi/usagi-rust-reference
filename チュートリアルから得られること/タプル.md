## タプル

file: [tuple.rs](.src/tuple.rs)

```rust
fn main( )
{
  let a: ( u8, f32, &str ) = ( 123, 4.5e+6, "789" );
  println!( "{}", a );
}
```

### 得られること

1. `(type0, type1, type2)`と記述するとタプル ( tuple ) 型となるようだ。
1. タプルは要素の型が `format!` 可能ならタプルごと `format!` できるようだ。
